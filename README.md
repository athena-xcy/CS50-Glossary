# Glossary 术语表

## 需求

1. 移动端列表的屏幕适配
2. 删除术语
3. 后台管理页面

## 简介

术语表前端，基于next.js + ant.design + mobx实现

![interface.png](https://i.loli.net/2019/09/23/N5SOlRMDkEPGYgT.png)

## 配置开发环境

```
npm install
API_ENDPOINT=... # 服务器url, 如http://localhost:5000
echo API_ENDPOINT=$API_ENDPOINT >.env
npm run dev
```

在浏览器打开`localhost:3000`
